Covid-19 Time Series Anomaly Detection

In the midst of Covid-19, the evolving Covid-19 mutations have been deadly. This research was conducted to see if Covid-19 symptom resolution data can be used to predict mutations before they happen. The methodology was to see a spike in Anomalies during the time the mutation was actually reported. An enhanced PCA LSTM Autoencoder was the novel approach to this problem and outperformed all the other baseline models. The paper publication for this research is coming soon.

Aerial Image Processing

Parachute location tracking can be a difficult task. GPS tracking can be risky because opponents can cause signal interfence and jam the signals. In this project, we look to perform semantic segmentation to classify the pixels in the image. Once the pixels are classified, a neural network will be trained to predict the change in location between two Class Pixel matrices.

Predicting Academic Performance

In this paper we fit models for the prediction of academic outcomes and identified their potential concerns and shortcomings. For the Student Performance Data Set we found that both Linear Regression and Random Forest are seemingly ineffective for meaningful prediction on Math data, while Random Forest is the most effective on Language data. Further, we identified that Principal Component Analysis did not provide any tangible benefit over the use of the raw data.

Email Text Classification App

During my internship at Fidelity Investments (FCAT), I worked on a email text classification problem. There was a large class imbalance issue, so I applied some random sampling methods to try to resolve the issue. There were around 30 classes in the dataset. I then created a front end and back end using Flask/HTML. The user interface allowed the User to enter various inputs (such as Embedding/TF-IDF vector, etc.) to train the classifier and they could view the Class Distribution using T-SNE dimension reduction, and various other plots/charts.

There is no portfolio for this work due to confidentiality reasons

Movie Review Text Classification

Obtained a set of Movie Reviews which was classified as Spam and Not spam. I used a TF-IDF vectorizer to transform the data and also trained various classifiers, but the MLP sklearn neural network performed the best on this dataset. We also used this trained model to predict on Spotify tweets which we stored on MongoDB.
